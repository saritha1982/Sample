SonarQube

SonarQube (formerly known as Sonar) is an open source Platform used by development teams to manage source code quality. It collects and analyzes source code, measuring quality and providing reports for the projects.
SonarQube quantifies the quality of your code in a way that is easily visible to every member of your team. The tool provides customizable dashboards that display quality metrics for your code each time the static analysis is performed. You can view seven different kinds of metrics such as unit test coverage, code duplication, and language-specific rule violations.
