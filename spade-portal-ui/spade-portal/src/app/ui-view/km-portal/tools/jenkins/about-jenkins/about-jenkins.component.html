Jenkins

Jenkins CI is a powerful application that allows continuous integration and continuous delivery of projects, regardless of the platform you are working on. You can integrate Jenkins with a number of testing and deployment technologies. It provides users with the ability to automate common software build processes throughout the development lifecycle.

We have a separate instance of Jenkins implemented to support custom development. This is integrated with Sonarqube, Rally, GitHub and Artifactory. The outcome of integration is for following reasons:

  • Github: This results in easy fetching of source code for immediate build and testing
  • Rally: To push overall results from the build and testing process
  • SonarQube: To fetch results of quality of the software
  • Artifactory: To push the software assets for customer download
