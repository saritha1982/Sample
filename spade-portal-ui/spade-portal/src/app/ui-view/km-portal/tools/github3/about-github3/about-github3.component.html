A source code repository is a file archive and web hosting facility where a large amount of source code, for software development is kept either publicly or privately. Github enterprise has been chosen as Source code Repository. GitHub is a Web-based Git repository hosting service, which offers all of the distributed revision control and source code management functionality of Git. It also provides access control and several collaboration features such as bug tracking, feature requests and task management. Github provides multiple integrations in the form of the web hooks to send notifications and alerts.
