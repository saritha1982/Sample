Artifactory

Artifactory is a version control for your binary artifacts (jar/war files, etc). Artifactory can be used with the following technologies Maven, Gradle and Ivy and using several different CI Servers such as Jenkins, TeamCity and Bamboo.

Artifactory is available for both internal and external consumption. This will be evolved or replaced over time to accommodate more automated distribution capabilities.

For More info on Artifactory : https://engit.cisco.com/#/ci/artifactory/index
